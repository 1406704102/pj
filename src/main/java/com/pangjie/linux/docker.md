### 镜像命令
    查看本地镜像列表: docker images
    查找某个仓库镜像: docker search <XXX>
    获取某个镜像: docker pull <XXX>
    参看某个镜像: docker inspect <XXX>
    删除某个镜像: docker rmi <XXX>
    
### 容器命令
    列出运行中容器: docker ps
    列出所有容器: docker ps -a
    停止,启动,结束,重启容器: docker stop,start,kill,restart
